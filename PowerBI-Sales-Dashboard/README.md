## Overview
This project features an interactive **Sales Performance Dashboard** developed using **Power BI Desktop** as part of a hands-on Power BI workshop. The dashboard provides insights into sales, cost, and profit across regions, customers, products, and time periods.

## Business Objectives
- Monitor overall sales, cost, and profit performance
- Identify top-performing countries and customers
- Analyze product-wise and sales-manager-wise sales
- Track quarterly and yearly sales trends

## Dashboard Highlights
- KPI cards for Total Sales, Cost, and Profit
- Sales by Country analysis
- Customer Type segmentation
- Product-wise sales breakdown
- Sales Manager performance comparison
- Year and Quarter-based trend analysis

## Tools & Technologies
- Power BI Desktop
- DAX (Measures & Calculations)
- Excel (Source Data)
- Data Modeling & Visualization

## Data Source
Workshop-provided sample sales data used strictly for learning and portfolio purposes.

## Key Learnings
- Designing business-focused dashboards
- Writing DAX measures for KPIs
- Using slicers, filters, and drill-downs
- Structuring dashboards for executive reporting

## Note
This dashboard was created as part of a **Power BI Workshop** to demonstrate practical Power BI and data visualization skills.
